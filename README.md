
## 📋 Installation Guide

### Windows Deployment
```batch
# Extract to preferred location
Extract-Archive setup-assistant.zip C:\TModLoader-Enhanced\

# Run as Administrator
Right-click setup.exe → "Run as Administrator"

# Follow wizard
1. Select installation directory (default: C:\Games\TModLoader)
2. Choose mod repository location
3. Configure network settings
4. Complete automated setup
```

### Linux Deployment
```bash
# Extract archive
tar -xzf setup-assistant.tar.gz -C ~/tmodloader-enhanced/

# Make executable and run
chmod +x setup.sh
sudo ./setup.sh

# Follow interactive prompts
# Launch via desktop shortcut or: tmodloader-enhanced
```

### macOS Deployment
```bash
# Extract and install
unzip setup-assistant.zip
sudo ./setup.sh

# Grant necessary permissions in System Preferences
# Launch from Applications folder
```
